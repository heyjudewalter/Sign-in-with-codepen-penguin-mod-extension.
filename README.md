How to use:

1. Download server.py from this repository.
2. Run the server.py through python it will do localhost on port 8080 you can change it to a different port if you need to.
3. Download the Codepen authentication.pmp from this repository.
4. Go to penguinmod editor: https://studio.penguinmod.com/editor.html
5. At the top of the website there should be a button that says file, click it.
6. Then click "load from this computer" below.
7. Upload the Codepen authentication.pmp
8. Then open the project in fullscreen by seeing some arrows at the right side of the screen.
9. Start the project by clicking the blue flag.
10. You should see some text READ IT. Hit space on the project to hide the text, show the sign in with codepen authentication button, and copy this github repo to the clipboard.
11. And ummmm do things.
